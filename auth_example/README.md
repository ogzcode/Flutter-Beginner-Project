# Auth Example
Flutter da auth işlemlerini ve buna bağlı yönlendirme mantığını öğrenmek için geliştirdiğim küçük uygulama.

## Kullandığım Teknolojiler
* Node.js
* Express.js
* Prisma 
* GetX
* Flutter
* Shared Preferences
* Dio

# Kullanım
```
cd backend 
npm i
npm start


cd mobile 
flutter pub get
flutter run
```
